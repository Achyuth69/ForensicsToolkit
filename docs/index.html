All-in-One Interface

Every forensic module in a single unified Qt6 application. No switching between tools.

AI-Powered Analysis

📄

Court-Ready Reports

Export findings as professional PDF, HTML, or JSON reports with chain-of-custody, case metadata, and executive summaries.

Hash Verification

MD5, SHA-1, SHA-256 hashing with OpenSSL. Verify evidence integrity throughout the investigation lifecycle.

YARA Detection

Load custom YARA rule sets for malware detection. Built-in heuristics catch packed PEs, shellcode, and extension mismatches.

Deep PCAP Analysis

Native PCAP parser reconstructs TCP streams, DNS queries, HTTP sessions, and flags suspicious C2 traffic patterns.

System Requirements

Runs on any modern Windows machine

Minimum

  • Windows 10 64-bit (1909+)
  • 4 GB RAM
  • 500 MB disk space
  • 1280×720 display

Recommended

  • Windows 11 64-bit
  • 16 GB RAM
  • SSD with 10 GB free
  • 1920×1080 display

Dependencies

  • Included in installer
  • Qt 6.11 runtime (bundled)
  • OpenSSL 4.x (bundled)
  • No .NET or Java required
